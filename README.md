## React + TypeScript Todo App

This is a small **React + TypeScript** project that shows a modern, responsive todo list UI.

### Features
- **Add todos** using a simple form with an input and an "Add Todo" button.
- **List of todos** displayed in a clean card-style layout.
- **Delete button (UI only)** is shown for each todo item, ready for you to hook your own logic.
- **Responsive design** that works on desktop and mobile (the form and list adapt to small screens).

### Tech Stack
- **React 18**
- **TypeScript**
- **Create React App (CRA)**
- **Plain CSS** for styling and responsiveness

### How to Run
1. Install dependencies:
   - `npm install`
2. Start the dev server:
   - `npm start`
3. Open in your browser:
   - `http://localhost:3000`

### How to Build
- Create a production build:
  - `npm run build`

You can extend this project by adding real delete logic, editing todos, or persisting data to localStorage or an API.
